# 07_Презентация

Слайды и сопровождающие материалы для защиты (Phase 7).

## Структура

- `_план_слайдов.md` — поэтапный план 12-15 слайдов.
- `Защита_Катальшов.pptx` — презентация.
- `Доклад.md` — полный текст доклада (хронометраж 7 минут).
- `QA_подготовка.md` — ≥ 20 вопросов комиссии и заготовленные ответы.
- `figures/` — иллюстрации (общие с ПЗ + специфичные слайдовые).
- `демо.mp4` (опционально) — 30-сек демо симулятора.
- `Раздаточные.pdf` (опционально).

## Целевая длительность

- Доклад: 7 минут.
- Вопросы: 3 минуты.

## Проверки

- [ ] 3 репетиции с замером времени.
- [ ] Backup на USB + распечатанный PDF.
- [ ] Демо проверено на оборудовании защиты.
