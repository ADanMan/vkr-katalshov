# 06_ПЗ — Пояснительная записка

Главы и финальный `.docx` ВКР (Phase 6).

## Структура

- `draft/` — Markdown-черновики глав:
  - `00_Реферат.md`
  - `01_Введение.md`
  - `02_Глава1_Аналитический_обзор.md`
  - `03_Глава2_Метод.md`
  - `04_Глава3_Реализация.md`
  - `05_Глава4_Эксперимент.md`
  - `06_Заключение.md`
  - `07_Приложения.md`
- `figures/` — иллюстрации Р1-Р17 (Mermaid-PNG, графики, схемы).
- `Литература.md` → `Литература.docx` — список источников по ГОСТ 7.0.5-2008.
- `ВКР_Катальшов.docx` — итоговая ПЗ.

## Целевой объём

65-75 страниц основного текста + приложения.

## Стиль

Times New Roman 14, межстрочный 1.5, поля 30/10/20/20 мм.

## Проверки перед сдачей

- [ ] Все рисунки/таблицы пронумерованы и подписаны.
- [ ] Все источники из `Литература.docx` цитируются в тексте, и наоборот.
- [ ] Реферат ≤ 1 страницы.
- [ ] Содержание автогенерируется в Word и совпадает со структурой.
- [ ] Антиплагиат ≥ 70% оригинальности.
- [ ] Прогон через `humanizer`-skill завершён.
