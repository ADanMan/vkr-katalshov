# 01_Источники

Здесь собрана литература, нормативка и аналоги под ВКР. Структура:

- `01_ГОСТ/` — национальные/межгосударственные стандарты (PDF + краткий конспект к каждому).
- `02_ФГОС/` — ФГОС 12.03.01, СУОС МГТУ, таблица компетенций (УК/ОПК/ПК), маппинг на ВКР.
- `03_МГТУ/` — методички и пособия Бауманки, шаблон ПЗ К2, Положение о ВКР бакалавра МГТУ.
- `04_Литература/` — научные статьи и книги, по тематикам:
  - `01_FLV_RV/` — Runtime Verification, STL, MTL, Maler, RTAMT, Breach.
  - `02_Метрология/` — теория измерений, ИИС, метрологическая надёжность.
  - `03_LabVIEW_TestStand/` — экосистема NI: TestStand SA, VI Analyzer, Requirements Gateway.
  - `04_FSM_Timed/` — конечные/тайм-автоматы, формальные методы, model checking.
  - `05_DSL_формализация/` — DSL, IDEF, OMG SysML, ISA-95.
- `05_Аналоги/` — документация и описания инструментов-аналогов FLV.

## Принципы именования файлов

- Книги/статьи: `Автор_Год_КраткоеНазвание.pdf` (`Maler_2004_MTL.pdf`).
- ГОСТы: `GOST_NNNN_YYYY_КраткоеНазвание.pdf` (`GOST_7.32_2017_Otchet_NIR.pdf`).
- Все имена ASCII (без пробелов и кириллицы), чтобы не ломать BibTeX-ключи и git-хук на Windows.

## Мастер-библиография

`BIBLIO.bib` (создаётся в Phase 1) — единая BibTeX-база. Каждая запись имеет ключ вида `авторгод_шорт`.
