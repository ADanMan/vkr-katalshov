# 04_Литература / 02_Метрология

Все PDF и метаданные русскоязычных статей получены с CyberLeninka (cyberleninka.ru); поля авторов, журналов, годов и страниц извлечены из meta-тегов `citation_*` и `eprints.*` страниц публикаций (дата обращения 2026-05-03). ## Что не покрыто этой партией (deferred)

- Журнал «Измерительная техника» (ВНИИМС, https://www.izmt.ru/) — закрытый платный доступ к статьям. Целевые статьи по метрологическому обеспечению ИИС искать через ЭБС МГТУ или РГБ.
- Диссертации (`disser.rsl.ru`) — поиск отложен; найден кандидат «Обеспечение метрологической исправности ИИС» (специальность ВАК 05.11.16). Если потребуется углубление — отдельный subagent.
- Журнал «Вестник МГТУ им. Н.Э. Баумана. Серия Приборостроение» — на vestnikprib.bmstu.ru есть статья «Верификация знаний бизнес-процессов»; не покрыто этим прогоном (не точно в теме измерений). Кандидат для второй партии.
