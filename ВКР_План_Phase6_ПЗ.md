## 1. Цель

Подготовить полный текст ВКР объёмом 65-75 страниц (без приложений) в формате `.docx`, оформленный по ГОСТ 7.32-2017 + шаблон К2 Бауманки (если получим). Готов к проверке антиплагиатом и сдаче на кафедру.

## 3. Структура каждой главы

### 3.1 Введение

- Актуальность (проблема семантического разрыва, цена ошибки в ИИС)
- Цель работы
- Задачи (5 задач из исходного плана)
- Объект исследования: ИИС, исполняющие программные сценарии
- Предмет исследования: методы функционально-логической верификации
- Научная новизна (3-4 пункта)
- Практическая значимость (4-5 пунктов)
- Методы исследования
- Структура работы

### 3.2 Глава 1 «Аналитический обзор»

- 1.1 Информационно-измерительные системы и метрологическая надёжность
- 1.2 Программные модели измерительных протоколов
- 1.3 Существующие подходы к верификации (TestStand SA, VI Analyzer, RTAMT, Breach, Requirements Gateway)
- 1.4 Обзор формальных методов: STL, MTL, Timed Automata
- 1.5 Семантический разрыв и обоснование необходимости FLV
- 1.6 Постановка задачи

### 3.3 Глава 2 «Метод и формальная модель»

- 2.1 Концепция функционально-логической верификации
- 2.2 DSL для нормативной модели измерительного процесса
- 2.3 Timed FSM как формальная основа
- 2.4 Алгоритм сопоставления (sequence/timing/predicate matchers)
- 2.5 Каталог нарушений
- 2.6 Интеграция с ИИС: контур, online/offline режимы

### 3.4 Глава 3 «Программная реализация»

- 3.1 Архитектура модуля FLV
- 3.2 Структура симулятора стенда
- 3.3 Детали реализации matchers
- 3.4 3D-визуализация и UI
- 3.5 Тестирование и метрики качества кода

### 3.5 Глава 4 «Экспериментальное исследование»

- 4.1 Методика эксперимента
- 4.2 Описание стендов S1 и S2
- 4.3 Каталог инъекций нарушений
- 4.4 Результаты и метрики
- 4.5 Статистическая обработка
- 4.6 Обсуждение результатов

### 3.6 Заключение

- Решённые задачи (mapping на 5 задач из введения)
- Полученные результаты
- Научная новизна (повтор)
- Практическая ценность (повтор)
- Перспективы развития

## 5. Оформление по ГОСТ 7.32-2017 (ключевое)

| Параметр | Значение |
|---|---|
| Шрифт основного текста | Times New Roman 14 pt |
| Межстрочный интервал | 1.5 |
| Поля | левое 30 мм, правое 10 мм, верхнее/нижнее 20 мм |
| Абзацный отступ | 1.25 см |
| Выравнивание | по ширине |
| Нумерация страниц | арабскими, в правом верхнем углу, начиная с введения, но счёт от титульного |
| Заголовки разделов | прописными, центрированные, без точки в конце |
| Заголовки подразделов | строчные с прописной первой, выравнивание по абзацу |
| Иллюстрации | подпись внизу: «Рисунок N — Название» |
| Таблицы | подпись сверху: «Таблица N — Название» |
| Ссылки на источники | в квадратных скобках [N] |
| Список источников | по ГОСТ 7.0.5-2008, нумерация в порядке упоминания |

## 7. Критерии завершения

- Объём 65-75 стр. без приложений.
- Все источники в библиографии цитируются в тексте; и наоборот.
- Реферат укладывается в 1 страницу.
- Антиплагиат ≥ 70% оригинальности.
- Все рисунки и таблицы пронумерованы и подписаны.
- Содержание автогенерируется в Word и совпадает со структурой.
- Файл `06_ПЗ/ВКР_Катальшов.docx` готов к печати.
