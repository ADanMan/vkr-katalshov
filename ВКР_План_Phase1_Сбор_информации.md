## 1. Структура папки источников

```
01_Источники/
├── _README.md                  ← список всех источников + индекс
├── 01_ГОСТ/                    ← федеральные ГОСТы (полные тексты)
├── 02_ФГОС/                    ← ФГОС 12.03.01 + учебный план/компетенции

├── 04_Литература/              ← научные статьи, книги, диссертации
│   ├── 01_FLV_RV/              ← runtime verification, STL, MTL
│   ├── 02_Метрология/          ← теория измерений, ИИС, метрологическая надёжность
│   ├── 03_LabVIEW_TestStand/   ← документация и публикации NI
│   └── 04_FSM_Timed/           ← конечные/тайм-автоматы, формальные методы

└── BIBLIO.bib                  ← BibTeX-файл (мастер-библиография)
```

Каждый файл-PDF переименовываем в `Автор_Год_КраткоеНазвание.pdf` (ASCII, без пробелов).

## 3. Метод сбора (как именно)

### Шаг 1.1 — ГОСТы (1 день)

Скрипт: `scripts/fetch_gosts.py` (создаём в Phase 2 при необходимости). Сейчас — вручную через `WebFetch` с docs.cntd.ru, allgosts.ru, internet-law.ru. Сохраняем PDF в `01_Источники/01_ГОСТ/`. Для каждого ГОСТа — короткий конспект (2-4 предложения) в `_README.md`.

### Шаг 1.2 — ФГОС и компетенции (0.5 дня)

Скачиваем ФГОС 12.03.01 (приказ, действующая редакция). Выписываем УК/ОПК/ПК-компетенции в таблицу `02_ФГОС/competencies.md`. Под каждую компетенцию — мэппинг «как закроет ВКР».

### Шаг 1.3 — Бауманские методички (0.5 дня)

Парсим открытые сайты `iu4.ru`, `bmstu.ru`, поиск на `lib.bmstu.ru`. Если шаблон ПЗ К2 закрытый — задача Q2 в открытых вопросах: попросить Данилу.

### Шаг 1.4 — Литература (3 дня)

Поиск через Google Scholar, Semantic Scholar (API), arXiv, eLibrary. Для каждой найденной работы:
- Сохранить PDF в соответствующую подпапку.
- Заполнить запись в `BIBLIO.bib` (BibTeX).
- Краткий конспект (3-7 предложений) в `_README.md` с тегом «зачем нужна».

Если статья платная и нет доступа — пометить в `_README.md` как `[CLOSED]` и отдать запрос Даниле для скачивания через ЭБС/университет.

### Шаг 1.5 — Аналоги (1 день)

Для каждого инструмента из таблицы 2.4: скриншот UI или схема функциональности, перечень фич, что из этого закрывает FLV-задачу, что — нет. Результат: `01_Источники/05_Аналоги/_сравнение.md` (таблица сравнения, готовая к перенесению в обзорную главу ВКР).

### Шаг 1.6 — Сборка отчёта Phase 1

Финальный артефакт: `01_Источники/Phase1_отчёт.md`. В нём:
- Список всех собранных источников (с типами, страничным охватом, статусом).
- Карта релевантности (какой источник работает на какую главу/раздел ВКР).
- Список **пробелов** (что не нашли — например, нет открытой работы про FLV в области ИИС → это и есть наша новизна).
- Чек-лист: 25+ источников, все ГОСТы из таблицы 2.1, ФГОС, минимум 5 аналогов, минимум 8 работ по STL/RV, минимум 5 русских статей по метрологии.

## 4. Критерии выхода из Phase 1

Phase 1 закрывается, когда:
- ✅ Все обязательные ГОСТы (N01-N06, N10) скачаны и законспектированы.
- ✅ ФГОС 12.03.01 — текст + таблица компетенций.
- ✅ ≥ 25 уникальных научных работ в `BIBLIO.bib` с заполненными полями.
- ✅ Сравнительная таблица аналогов готова.
- ✅ Конспект `Phase1_отчёт.md` доступен и читается за 15 минут (executive summary).
- ✅ Шаблон ПЗ К2 либо найден, либо явно зафиксирован как пробел (Q2).
