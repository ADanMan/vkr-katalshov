# agents/ — переиспользуемые subagent-шаблоны для ВКР

Каждый файл — готовый prompt для запуска через `Agent` tool с `subagent_type: general-purpose`. Подставляй переменные `{{...}}` в свой контекст.

## Как использовать в Cowork / Claude Code

```
Agent(
  description="<коротко 3-5 слов>",
  subagent_type="general-purpose",
  prompt="<содержимое нужного файла из agents/, с подставленными переменными>"
)
```

## Принципы

1. **Минимум переменных**: каждый агент берёт максимум 2-3 параметра, остальное — фиксированный контекст ВКР.
2. **Жёсткие выходные форматы**: каждый агент знает, куда положить артефакты и в каком формате вернуть отчёт.
3. **Лимит ответа**: 200-400 слов, чтобы не забивать главный контекст.
4. **Compliance-flag**: если агент находит источники с правовыми рисками — обязательно отмечает в отчёте.
5. **Никаких выдуманных фактов**: если данные не верифицируются — пометка `[VERIFY]` или `[NOT FOUND]`.
