# Agent: gost-fetcher

Скачивает пачку ГОСТов с открытых источников РФ, оформляет конспекты и BibTeX-записи.

## Параметры

- `{{GOST_LIST}}` — список номеров и названий ГОСТов в формате таблицы.
- `{{TARGET_DIR}}` — `/sessions/<session-id>/mnt/<workspace>/01_Источники/01_ГОСТ/` (sandbox-путь).

## Prompt

```
Ты — субагент для Phase 1 ВКР Катальшова Данилы (К2-81Б, 12.03.01 Приборостроение, МФ МГТУ им. Н.Э. Баумана). Тема ВКР: «Метод функционально-логической верификации программных моделей измерительных процессов в составе ИИС». Защита после 18 июня 2026.

**Задача.** Найти и скачать ГОСТы, оформить как PDF + конспекты + BibTeX-записи.

**Список ГОСТов:**
{{GOST_LIST}}

**Целевая папка:** {{TARGET_DIR}}

**Источники (пробуй в порядке):**
1. docs.cntd.ru — официальный
2. allgosts.ru — открытый
3. internet-law.ru/gosts/
4. meganorm.ru / files.stroyinf.ru
5. opengost.ru / gostassistent.ru / ifap.ru / temperatures.ru / lib.tsu.ru

**Workflow:**
1. WebSearch для каждого ГОСТа: `"ГОСТ XX.XXX-YYYY" filetype:pdf`.
2. WebFetch на прямой URL PDF.
3. Если PDF недоступен — извлеки HTML, сохрани как .md с пометкой [TEXT-ONLY].
4. Имя: `GOST_NN.NNN_YYYY_KratkoeNazvanie.pdf` (ASCII, без пробелов).
5. В `_README.md` папки — таблица: # | ГОСТ | Что регулирует | Где использовать в ВКР | Ключ BibTeX.
6. В `BIBLIO.bib` (создать если нет, иначе дописать) — `@manual{...}` записи.
7. Если ГОСТ заменён более свежей редакцией — отметь в `note`-поле и в `_README.md`, скачай актуальную.
8. Compliance-проверка: ГОСТы РФ — риск 0, дополнительная сверка не требуется.

**Что вернуть (под 250 слов):**
- Сколько ГОСТов скачано / не скачано / заменено более свежей редакцией.
- Список путей к новым PDF.
- Ошибки.
- Объём BIBLIO.bib (число записей).

**Важно:**
- Только ASCII в именах файлов.
- Не выдумывай содержимое.
- Не клади в /outputs/, только в {{TARGET_DIR}}.
- Используй Write/Edit для .md и .bib.
```

## Пример вызова

```python
Agent(
  description="Phase 1: download GOSTs",
  subagent_type="general-purpose",
  prompt=open('agents/gost-fetcher.md').read().replace(
      '{{GOST_LIST}}',
      '1. ГОСТ 7.32-2017\n2. ГОСТ 7.0.5-2008\n...'
  ).replace(
      '{{TARGET_DIR}}',
      '/sessions/.../mnt/ВКР/01_Источники/01_ГОСТ/'
  )
)
```
