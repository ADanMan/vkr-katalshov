# Структурный референс из ifsmirnov/bachelor-diploma

LaTeX-шаблон бакалаврского диплома, форк русского `Russian-Phd-LaTeX-Dissertation-Template`. Сам шаблон не используем (финал — `.docx`), но **порядок разделов и принципы организации файлов** — берём.

Репо: <https://github.com/ifsmirnov/bachelor-diploma>. Лицензия CC BY 4.0 (атрибуция авторов оригинала: Юлия Мартынова, A. Akinshin et al.).

## Порядок разделов в шаблоне

```
dissertation.tex (главный)
├── Титульный лист
├── Оглавление
├── Введение
├── Глава 1
├── Глава 2
├── Глава 3
├── (опц.) Заключение
├── (опц.) Список сокращений
├── (опц.) Словарь терминов
├── Список литературы
├── (опц.) Списки таблиц и иллюстраций
└── (опц.) Приложения
```

Совпадает с требованиями ГОСТ 7.32-2017 (см. `gost_7.32_structure.md`) — используем как канву.

## Что переносим в наш .docx-workflow

### 1. Файловая организация

В шаблоне каждая глава — отдельный `.tex`-файл, который потом включается в главный документ. У нас аналог — отдельные `.md`-файлы в `06_ПЗ/draft/`:

```
06_ПЗ/draft/
├── 00_Реферат.md
├── 01_Введение.md
├── 02_Глава1_Аналитический_обзор.md
├── 03_Глава2_Метод.md
├── 04_Глава3_Реализация.md
├── 05_Глава4_Эксперимент.md
├── 06_Заключение.md
└── 07_Приложения.md
```

Сборка через pandoc (см. `scripts/md_to_docx_gost.py`).

### 2. Папка для библиографии

В шаблоне — `biblio/`. У нас — корневой `BIBLIO.bib` плюс `06_ПЗ/draft/Литература.md` для финального форматирования по ГОСТ.

### 3. Папка для изображений

В шаблоне — `images/` (общие) + `Dissertation/images/` (специфичные). У нас:
- `02_Спецификация/figures/` — диаграммы метода.
- `03_Симулятор/viz/` — скриншоты симулятора.
- `05_Эксперименты/figures/` — графики эксперимента.
- `06_ПЗ/figures/` — все картинки, отобранные для финальной ПЗ (копии или симлинки).

### 4. Управление компиляцией кусков

В шаблоне — `inclusioncontrol.tex` для выборочной сборки. У нас — отдельные `.md`-черновики собираются в `.docx` независимо, а склейка делается на финальном этапе. Это даёт гибкость: можно собрать главу 4 в `.docx`, отнести научруку, получить правки, переделать — без полной пересборки.

### 5. Размер страницы и шрифт

Шаблон: A4, 14pt. У нас: A4, TNR 14, межстрочный 1.5, поля 30/10/20/20 — в точности соответствует Положению МГТУ для бакалавриата.

## Что НЕ переносим

- LaTeX-пакеты, стили, классы — не нужны (мы в .docx).
- PSCyr (русские шрифты) — у нас Word с TNR.
- BibTeX-стили из шаблона — у нас CSL `gost-r-7-0-5-2008.csl` для pandoc.
- Раздел «Synopsis» (автореферат) — не нужен для бакалавра.
- Папка `Presentation` (Beamer) — мы делаем `.pptx`.

## Готовые PDF в репо шаблона

В репо `ifsmirnov/bachelor-diploma` лежат скомпилированные PDF (`diploma.pdf`, `dissertation.pdf`, `diploma_black.pdf`). Можно скачать как референс оформления (как должна выглядеть страница), но **не копировать содержимое** — только смотреть форматирование.
