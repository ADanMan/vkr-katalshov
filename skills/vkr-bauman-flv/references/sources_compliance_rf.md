# Compliance-проверка источников (РФ)

При работе над ВКР в России следует следить, чтобы источники не нарушали 4 категории законодательных ограничений. **Этот файл — не юридическое заключение**, но рабочий чек-лист для самоконтроля.

## 1. Категории риска

| # | Категория | Реестр | Что нельзя |
|---|---|---|---|
| 1 | Иностранные агенты (физлица, медиа, организации) | Минюст РФ | Цитировать публикации СМИ-иноагентов **без маркировки** «настоящий материал/сообщение создан и (или) распространён иностранным агентом ...». |
| 2 | Нежелательные организации | Генпрокуратура + Минюст | Использовать издания/публикации в качестве источников — **запрещено**. Распространение материалов = адм. ответственность (КоАП 20.33). |
| 3 | Экстремистские материалы | Минюст РФ | Цитировать или использовать — **запрещено**. УК 282.1 за организацию, КоАП 20.29 за распространение. |
| 4 | LGBT-движение (с 30.11.2023) | Решение ВС РФ | Не цитировать материалы движения и не позиционировать положительно. |

## 2. Чек-лист для каждой добавляемой записи в `BIBLIO.bib`

```
Перед добавлением записи проверь:

[ ] Автор — НЕ значится в реестре иноагентов (https://minjust.gov.ru/ru/activity/directions/998/).
[ ] Издатель/издание — НЕ в списке нежелательных организаций (https://minjust.gov.ru/ru/activity/directions/996/).
[ ] Содержимое — НЕ в федеральном списке экстремистских материалов (https://minjust.gov.ru/ru/extremist-materials/).
[ ] URL — НЕ ведёт на:
    • Sci-Hub (для публичной библиографии — указывай исходного издателя)
    • Иноагент-СМИ (Meduza, Дождь и т.п.)
    • Заблокированные ресурсы Роскомнадзором (если ВКР проходит проверку из РФ-сети)
[ ] Тема публикации — техническая/научная, без политики (для STL/RV/метрологии — почти всегда ОК).
```

## 3. Безопасные категории (риск 0)

- ГОСТы и нормативные акты РФ.
- Учебники и методички ВУЗов РФ (МГТУ, МИФИ, СПбГУ и пр.).
- Статьи в журналах ВАК и РИНЦ.
- Диссертации из РГБ/eLibrary.
- Иностранные академические публикации в Springer / Elsevier / IEEE / ACM / Wiley / Taylor & Francis.
- Препринты arXiv по техническим темам.
- Open-source инструменты под OSI-лицензиями (MIT, BSD, Apache, GPL, LGPL, CC).
- Документация публичных проектов с GitHub.

## 4. Что НЕ использовать

- Wikipedia (академически не принимается; легально — без проблем).
- Habr — допустимо **если** это статья от верифицированного автора и оформлена по ГОСТ 7.0.5; не основной источник.
- YouTube, блоги, форумы — не цитируются.
- Telegram-каналы — не цитируются.
- Sci-Hub URL — не указывать в библиографии (указывай исходного издателя).
- Иноагент-СМИ и нежелательные организации — полностью исключать.

## 5. Рекомендуемая структура библиографии для ВКР МГТУ

| Категория | Доля |
|---|---|
| ГОСТы и норм. акты РФ | 15-25% |
| Русскоязычные учебники, статьи, диссертации | 25-40% |
| Иностранные академические работы (Springer/Elsevier/IEEE/ACM/arXiv) | 35-60% |
| Электронные ресурсы (документация, repos) | до 10% |
| Wikipedia, блоги, неакадемика | **0%** |

## 6. Когда применять этот чек-лист

- **Перед** каждым коммитом, который добавляет ≥ 5 записей в `BIBLIO.bib`.
- **При завершении Phase 1** — пройди весь BIBLIO.bib через чек-лист.
- **Перед сдачей ПЗ** — финальная сверка с актуальными реестрами (они обновляются).

## 7. Если возник риск

1. Останови работу с источником.
2. Проверь актуальный статус через Минюст / Генпрокуратуру.
3. Если статус подтверждается → удалить из библиографии или (для иноагентов-СМИ при необходимости упоминания) — добавить маркировку и пометить в `note` BibTeX-записи.
4. Если сомневаешься → не используй. Найди аналог.

## 8. Источники нормативки

- ФЗ № 255-ФЗ от 14.07.2022 «О контроле за деятельностью лиц, находящихся под иностранным влиянием».
- ФЗ № 272-ФЗ от 28.12.2012 «О мерах воздействия...».
- ФЗ № 114-ФЗ от 25.07.2002 «О противодействии экстремистской деятельности».
- Постановление КС РФ от 14.07.2021 № 35-П.
- Решение ВС РФ от 30.11.2023 о международном движении LGBT.
