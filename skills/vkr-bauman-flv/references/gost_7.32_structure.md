# ГОСТ 7.32-2017 — структура отчёта о НИР, под нашу ВКР

ГОСТ 7.32-2017 «СИБИД. Отчёт о научно-исследовательской работе. Структура и правила оформления». Применяется для бакалаврских ВКР технического направления.

## Обязательные структурные элементы (порядок важен)

1. **Титульный лист.** Источник информации для регистрации работы. На нашем шаблоне (К4) — см. `bauman_template_guide.md`.
2. **Список исполнителей.** Для бакалавра-индивидуала обычно опускается или содержит только одного автора. Уточни у научрука перед сдачей.
3. **Реферат.** ОБЯЗАТЕЛЬНО: объём отчёта (страницы), число иллюстраций, таблиц, использованных источников, приложений. Затем перечень ключевых слов (от 5 до 15 слов или словосочетаний прописными через запятые), затем 1 абзац — суть работы (объект, цель, методы, результат, область применения). Лимит — 1 страница.
4. **Содержание.** Автогенерация в Word. Включает все заголовки разделов 1-3 уровней, заключение, список источников, наименования приложений.
5. **Перечень сокращений и условных обозначений.** Если в тексте используется > 5 нестандартных сокращений или символов. Включаем: ИИС, ПЗ, FLV, FSM, STL, MTL, RV, DSL, SCPI, VISA.
6. **Введение.** 4-6 стр. Содержит: актуальность, цель, задачи (5 шт., копируем из плана), объект, предмет, научная новизна, практическая значимость, методы исследования, краткое описание структуры работы.
7. **Основная часть.** Главы 1-4 (см. SKILL.md, §7).
8. **Заключение.** 2-3 стр. Главные итоги, оценка достижения цели, рекомендации, перспективы.
9. **Список использованных источников.** ≥ 25 (наша цель), нумерация в порядке упоминания.
10. **Приложения.** Обязательные и информационные.

## Оформление страницы

| Параметр | Значение |
|---|---|
| Бумага | A4 |
| Шрифт основного текста | Times New Roman 14 pt |
| Межстрочный интервал | 1.5 |
| Поле левое | 30 мм |
| Поле правое | 10 мм |
| Поле верхнее | 20 мм |
| Поле нижнее | 20 мм |
| Абзацный отступ | 1.25 см |
| Выравнивание основного текста | по ширине |
| Цвет шрифта | чёрный |
| Печать | односторонняя |

## Заголовки

- **Раздел (1-й уровень, главы):** прописными, центр, без точки в конце, нумерация арабскими цифрами начиная с «1». Шрифт TNR 14, **жирный**.
- **Подраздел (2-й уровень):** строчные с прописной первой, выравнивание по абзацу, нумерация формата «1.1». TNR 14, жирный.
- **Пункт (3-й уровень):** аналогично, нумерация «1.1.1». TNR 14, жирный.
- Перенос слов в заголовках запрещён.
- Между заголовком и текстом — пустая строка (или интервал, эквивалентный пустой строке).

## Нумерация страниц

- Арабскими цифрами, шрифт TNR 12, **в правом верхнем углу** без точки.
- Титульный лист считается, но НЕ нумеруется.
- Страницы реферата, содержания, перечня сокращений — нумеруются.
- Нумерация начинает быть видимой с введения (но счёт идёт от 1).

## Иллюстрации

- Подпись **снизу** иллюстрации: «Рисунок N — Название» (тире длинное, с пробелами).
- Сквозная нумерация по работе, либо в пределах раздела (`1.1`, `1.2`).
- Шрифт подписи — TNR 12, по центру.
- Обязательная ссылка в тексте: «...как показано на рисунке 3».
- Цветовая палитра: для печати — учитывать, что чёрно-белая печать всё ещё распространена.

## Таблицы

- Подпись **сверху** таблицы: «Таблица N — Название» (выравнивание влево, отступ 0).
- Сквозная или раздельная по разделам нумерация.
- Шапка — жирный шрифт, по центру; данные — TNR 12-14, по содержимому.
- Если таблица переносится — на новой странице повторить шапку и подписать «Продолжение таблицы N».

## Формулы

- Печатаются с новой строки, по центру, с отступом 1 пустая строка до и после.
- Нумеруются арабскими цифрами в круглых скобках, выравнивание номера — по правому краю: `(1.4)`.
- Расшифровка переменных — после формулы со словом «где», каждый символ — с новой строки.

Пример:

> dT/dt = (T_set − T) / τ + ξ(t),       (3.1)
>
> где T — текущая температура образца, °C;
> T_set — заданная температура, °C;
> τ — постоянная времени системы, с;
> ξ(t) — случайный шум сенсора, °C.

## Перечисления

- Маркером выступает дефис «−» (НЕ дефис-минус «-»), либо буквы строчной с круглой скобкой `а)`, `б)`, либо арабские цифры с круглой скобкой `1)`, `2)`.
- После маркера — пробел, затем строчная буква.
- В конце каждого пункта — точка с запятой; в конце последнего — точка.

## Ссылки на источники

- В тексте: `[N]` — номер по списку использованных источников. Допускается группировать: `[1, 5, 7-9]`.
- На иллюстрацию: «(рисунок 3)» или «...на рис. 3» — оба варианта допустимы, но в пределах работы соблюдай единый.
- На таблицу: «...в таблице 2 представлены...».
- На формулу: «...в соответствии с формулой (3.1)...».

## Приложения

- Каждое приложение — с новой страницы.
- Заголовок: «ПРИЛОЖЕНИЕ А» (буква русского алфавита, без Ё, З, Й, О, Ч, Ь, Ы, Ъ).
- Под заголовком — наименование приложения, например, «Полная DSL-спецификация».
- Иллюстрации, таблицы и формулы внутри приложения нумеруются: «Рисунок А.1», «Таблица Б.2», «(В.3)».

## Типовые ошибки оформления

- ✗ Дефис вместо тире в подписях рисунков и таблиц («Рисунок 1 - Название» вместо «Рисунок 1 — Название»).
- ✗ Знак умножения буквой «х» вместо «×».
- ✗ Кавычки разные («», "", “”) — должны быть «ёлочки» в основном тексте, „внутренние" — для вложенных.
- ✗ Английская «c» вместо русской «с» в сокращениях («ГОСТc» вместо «ГОСТс»).
- ✗ Неразрывные пробелы между числом и единицей измерения отсутствуют («150°C» — нужен `150 °C` с неразрывным пробелом).
- ✗ Список литературы оформлен «Mendeley APA» вместо ГОСТ 7.0.5.
